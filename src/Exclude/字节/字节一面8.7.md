Q2: 如何让其符合预期, 不使用let

```js
for (var i = 0; i < 5; ++i) {
  setTimeout(function (){
    console.log(i + ' ');
  }, 100);
}
```

解:

```js
for (let i = 0; i < 5; ++i) {
  setTimeout(function (){
    console.log(i + ' ');
  }, 100);
}
```

```js 立即执行函数
for (var i = 0; i < 5; ++i) {
    (function(i){
        setTimeout(function (){
            console.log(i + ' ');
        }, 100);
    })(i)
}
```
