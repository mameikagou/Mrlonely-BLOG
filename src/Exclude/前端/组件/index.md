
### 如何考量以及设计一个组件？

一点一点讲：

#### 通用组件：

1. 样式，在不同场景下的不同样式；比如一个button可能好几种样式，包括圆的方的，大小，颜色等等；
2. 交互，比如一个button，点击的时候有不同的效果，比如loading，比如不可点击等等；
3. 事件，浏览器事件的触发与阻止，与样式loading的交互等等；

- 高内聚低耦合：将相关逻辑放在一起，将
- 通用性：样式的通用性，允许用户自定义内容，比如表格项预留自己的内容
```tsx
const columns = [
  {
    title: '名称',
    dataIndex: 'name',
    width: 200,
    render(text) {
      return <em>{text}</em>;
    },
  },
];

<Table columns={columns} />;
```
