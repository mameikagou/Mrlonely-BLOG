
## 层缓存

### ❌ 不好的做法
COPY . .                  # 任何文件变化都会使缓存失效
RUN npm install           # 即使 package.json 没变，也要重新安装

### ✅ 好的做法
COPY package*.json ./     # 只有 package.json 变化才失效
RUN npm install          # 依赖没变就使用缓存
COPY . .                 # 源代码变化不影响上面的层
