# ⚡ Rainjs 前端逆袭：

## 👶 大一：死磕基础 (别人打游戏你在敲代码)
- **目标**：大一结束时，水平要超过大三的混子学长
- **核心基础 (地基打死)**
  - HTML5/CSS3 (Flex/Grid 布局写到吐)
  - **JavaScript ES6+ (这是命根子)**
    - 闭包、原型链、作用域 (必须极其熟悉，张口就来)
    - Promise/Async (不懂这个别投简历)
- **实战动作**
  - 扔掉教科书，看 MDN 文档
  - 仿写大厂静态页面 (如: 1:1 还原京东首页)

## 👦 大二：拉开差距 (弯道超车关键期)
- **目标**：**拿下第一份实习** (哪怕是 50 人小厂)
- **技术栈升级**
  - **Vue 3 全家桶** (Vue3 + TS + Vite + Pinia)
  - 或者 React (Hooks 深度应用)
  - **Node.js 基础** (了解后端怎么写接口，方便甩锅)
- **算法启动 (LeetCode)**
  - 每天 1 道 Medium，保持手感 (Hot 100)
- **简历级项目**
  - **必须上线！** (部署到阿里云/Vercel/cloudflare)
  - 复杂度要求：要有权限管理、虚拟列表、大文件上传等等基本配置。
- **行动点**
  - **大二下暑假：去实习！去实习！去实习！**
  - 积累真实业务经验，这是大三进大厂的门票

## 🐺 大三：收割大厂 (降维打击)
- **目标**：手里至少要有 1 段大厂实习经历
- **大三上：日常实习 (Daily Intern)**
  - 拿着大二的实习经历，投字节/美团/快手
  - 此时竞争对手很少，成功率极高
- **大三寒假：冲刺腾讯/阿里**
  - 准备暑期实习提前批
- **进阶深水区 (面试护城河)**
  - **前端工程化** (Webpack/Vite 插件开发)
  - **性能优化** (指标监控 SDK、首屏秒开)
  - **源码阅读** (Vue3 Reactivity 源码)

## 👑 大四：诸神黄昏 (Offer 决赛圈)
- **目标**：不只是拿 Offer
- **7-9月：秋招收官**
  - 此时你已经有 1 年+ 的大厂工龄
  - 面试基本是降维打击，和面试官聊业务痛点
- **10月-12月：开奖**
  - 开奖之后等着躺平
- **毕设**
  - 随便做一个，重心完全在搞钱和职业规划

## ❌ 绝对不要做的事
- **不要参加学校的水比赛** (除了icpc之外的比赛，对就业的含金量不如一段实习)
- **不要参与班干部/学生会/各种社团/校级组织/老师的实验室** (纯牛马过家家)
- **不要花时间学 JSP/jQuery** (直接入土)
