# 🚀 双非前端逆袭：四年全规划 (阿强亲历版)

## 👶 大一：低头看路，抬头做人
- **关键词：** 打基础、不挂科、信息差
- **核心任务**
  - 🍺 享受大学生活 (多交朋友，以后都是人脉)
  - 📚 基础语言 (C/C++/Python 选一个入个门)
  - 🚫 **底线：** 别挂科 (挂科影响推优，虽然我挂了，但你别学我)

## 👦 大二：梦的开始 (最关键的一年)
- **关键词：** 刷题、项目、第一份实习
- **技术栈启动**
  - HTML5 + CSS3 + JavaScript (ES6+)
  - 必须搞定一个能看的项目 (别做Todo List，做个管理后台)
- **算法起步**
  - LeetCode Hot 100 (每天一题，保持手感)
- **🎯 核心目标**
  - **找一份实习！** (哪怕是小公司/本地创业公司)
  - *目的：* 简历上不能是空白，这是大三冲大厂的跳板

## 👨‍🎓 大三：生死时速 (决定命运)
- **大三上：死磕大厂实习**
  - 投递：字节、美团、快手 (日常实习不卡学历，只看技术)
  - **战略目标：** 大三下之前，必须有一段大厂实习经历 (工牌是最好的通行证)
- **大三寒假：腾讯专项突击**
  - 💡 **双非本的“黄金窗口”**
  - 准备：寒假回来(3月)直接面腾讯暑期实习
  - *内幕：* 腾讯暑期实习对双非最友好，转正率高，这是进鹅厂最容易的一次
- **技术进阶**
  - Vue3 / React (深度原理)
  - 计算机网络 / 浏览器原理 (面试必问)

## 🎓 大四：收割 & 谈钱
- **大四上 (7月-9月)：秋招决战**
  - 拿着你的大厂实习背书，海投！
  - 重点攻克：美团、字节、阿里、京东
- **大四上 (11月-12月)：开奖 & 谈薪**
  - 此时Offer已发，开始谈钱
  - **Argue策略：** 用A厂的Offer去抬B厂的价 (私聊教你话术)
  - 🎓 **毕设：** 也就是走个过场，重心完全在工作

## 🏢 各大厂通关密籍 (点击看标签)
- **ByteDance (字节)**
  - ☠️ *难度：* S级 (双非地狱模式)
  - 🏷️ *特点：* 算法狂魔、疯狂考手写代码
- **Meituan (美团)**
  - 😄 *难度：* A级 (双非友好)
  - 🏷️ *特点：* 重视项目实战、务实、Vue大本营
- **Tencent (腾讯)**
  - 🎓 *难度：* A+级 (卡学历，但暑期实习有口子)
  - 🏷️ *特点：* 问得广、计算机基础(计网)问得深
- **Ali (阿里)**
  - 📜 *难度：* S级
  - 🏷️ *特点：* Java味重、P7以下不要(校招缩招严重)
