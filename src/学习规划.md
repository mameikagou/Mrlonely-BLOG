# 学习规划（前端 + AI/Agent）

## 目标岗位与范围
- 主方向：前端工程师 + AI 全栈/Agent 开发
- 近期主线：前端面试为主线，AI/Agent 为并行补强
- 产出形态：面试题笔记、项目面表达、AI 工具链与 Agent 实战总结

## 时间框架（默认）
- 周期：8 周
- 强度：每周 5 天，每天 1-2 小时
- 结构：
  - 1-4 周：前端基础 + 面试题主攻
  - 5-6 周：工程化与系统设计 + AI 工具链
  - 7-8 周：项目面表达 + Agent/AI 全栈方向整理

> 可根据实际面试时间窗口压缩为 4 周冲刺版（删减内容保留高频与短板）。
